# Clevo-Service-ASR
Java based servlet service for speech transcription

## Structure

index.html - GraphiQL
/graphql - Service endpoint

## Running
Compile and package (build/libs/Clevo-Service-ASR.war)

`./gradlew build`

Run locally on jetty (http://localhost:8080/Clevo-Service-ASR)

`./gradlew appRun`
